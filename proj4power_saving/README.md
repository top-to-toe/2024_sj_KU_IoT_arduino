## 전력 절약을 위한 전력 측정 장치 개발

콘센트에 연결된 장치의 소비전력을 측정하여 데이터베이스에 기록하는 IoT 장치를 제작하고, 각 장치 별 전력 소비를 웹을 통해 모니터링 할 수 있는 기능 구현.
